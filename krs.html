As an Intern at KickBack Rewards Systems, I got to learn the basics of web design and programming

  • HTML - HyperText Markup Language is the visible language of the internet. All the words you see on a webpage are a part of the HTML document.
  • CSS - Cascading Style Sheets are a way to apply formatting to an HTML document. All colors, font-sizes, and many other things are controlled with CSS.
  • Javascript - Javascript is the programming language of the web. It can make elements of a webpage responsive and interactive.
  • Markdown - Markdown is a simple way to write formatted text withought using a word processor. It is easy to learn and is used to format blog posts or other web content withought using HTML.
  • Ruby - Ruby is a programming language
