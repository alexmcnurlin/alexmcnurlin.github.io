As an Intern at KickBack Rewards Systems, I got to learn the basics of web design and programming

  • HTML - HyperText Markup Language is the language of the internet. All the words you see on a webpage are in the HTML document.
  • CSS - Cascading Style Sheets are a way to apply formatting to an HTML document. All colors, font-sizes, and many other things are controlled with CSS.
  • Javascript - Javascript makes webpages look and feel more alive. Javascript can make elements of a webpage respond to mouse movements and inputs.
